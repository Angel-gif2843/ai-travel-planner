# AI Travel Planner

An AI-powered web application that generates customized travel itineraries based on user destination, duration, and budget. The application leverages an LLM to dynamically generate structured schedules and render them on a clean, responsive web interface.

## 🛠️ Tech Stack

* **Backend:** Python (FastAPI, Uvicorn)
* **Frontend:** HTML5, CSS3, JavaScript
* **AI Integration:** Groq API (LLaMA 3)
* **Version Control:** Git, GitHub

---

## 🚀 Features

* **Custom Itineraries:** Generates day-by-day travel recommendations tailored to user inputs.
* **Budget Filtering:** Adapts the travel plan according to specified budget constraints.
* **Dynamic UI Rendering:** Parses raw AI text output into beautifully structured, easy-to-read timelines.
* **Secure Environment:** Keeps API credentials hidden using secure environment variable routing.
