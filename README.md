# AI Travel Planner

An intelligent, full-stack web application developed during my internship that generates customized travel itineraries based on user inputs such as destination, duration, and budget. The application leverages a large language model to dynamically parse structured travel schedules onto a clean web interface.

## 🛠️ Technologies Used
* **Backend:** Python, FastAPI, Uvicorn
* **Frontend:** HTML5, CSS3, JavaScript
* **AI Integration:** Groq API (LLM)
* **Version Control:** Git, GitHub

## 🚀 Features
* Dynamic form handling for travel constraints (destination, budget, days).
* Real-time streaming and structured formatting of AI-generated itineraries.
* Secure environment variable routing for protected credentials.
